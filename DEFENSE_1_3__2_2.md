## 1.3 — Простые итерации (Якоби) и Зейдель

**Суть**: решаем СЛАУ A·x = b итерационно. Приводим к виду x = αx + β делением каждого уравнения на свой диагональный элемент: α_ij = −a_ij/a_ii (j ≠ i), β_i = b_i/a_ii.

**Условие сходимости**: ||α||_∞ < 1 (максимальная строчная сумма модулей матрицы α). Обеспечивается диагональным преобладанием исходной A.

**Якоби**: x^(k+1) = β + α·x^(k). Все компоненты считаются из **старых** значений.

**Зейдель**: то же самое, но при вычислении x_i^(k+1) подставляем **уже обновлённые** x_1^(k+1), ..., x_{i-1}^(k+1). За счёт этого сходится быстрее.

**Оценка погрешности** (ключевой момент — формулы разные!):

- Якоби (формула 1.20): ε^(k) = ||α|| / (1 − ||α||) · ||x^(k) − x^(k-1)||_∞
- Зейдель (замечание §1.1.4): ε^(k) = ||C|| / (1 − ||α||) · ||x^(k) − x^(k-1)||_∞

где C — **строго верхнетреугольная** часть матрицы α (элементы α_ij при j > i, остальные нули). Знаменатель в обоих случаях одинаковый: (1 − ||α||). Разница только в числителе: ||α|| у Якоби, ||C|| у Зейделя. Поскольку ||C|| ≤ ||α||, оценка Зейделя жёстче.

Останов: ε^(k) < ε. Зейдель сходится примерно в 4 раза быстрее Якоби.

## 1.4 — Метод вращений Якоби (собственные значения симметричной матрицы)

**Суть**: симметричная матрица → находим λ и v такие, что A·v = λ·v. Идея: поворотами «убиваем» внедиагональные элементы — на диагонали остаются собственные значения.

**Алгоритм** (каждая итерация):
1. Ищем max|a_pq| среди внедиагональных элементов.
2. Угол: φ = (1/2)·arctg(2a_pq / (a_pp − a_qq)). Если a_pp = a_qq → φ = π/4.
3. Строим матрицу вращения U (единичная, но u_pp = u_qq = cos φ, u_pq = −sin φ, u_qp = sin φ).
4. A^(k+1) = U^T · A^(k) · U — выбранный элемент a_pq обнуляется.

**Почему работает**: сумма квадратов внедиагональных элементов строго убывает на каждой итерации.

**Собственные векторы**: V = U_0 · U_1 · ... · U_k (накопленное произведение всех вращений). Столбцы V — собственные векторы.

**Останов**: t(A) = √(Σ a_ij², i≠j) < ε.

**Проверка**: для каждого j вычисляем ||A·v_j − λ_j·v_j|| — должно быть ≈ 0. Плюс ортогональность: ⟨v_i, v_j⟩ ≈ 0.

## 1.5 — QR-алгоритм (собственные значения любой матрицы)

**Отличие от 1.4**: метод вращений — только для симметричных матриц. QR-алгоритм — для **любых**, включая несимметричные, и может находить **комплексные** собственные значения.

**QR-разложение** (метод Хаусхолдера): A = Q·R, где Q — ортогональная (Q^T·Q = E), R — верхняя треугольная. Матрица Хаусхолдера H = E − 2vv^T/v^Tv — отражение, обнуляющее всё под диагональю в одном столбце за раз. Применяем последовательно к каждому столбцу.

**QR-алгоритм**: A^(k) = Q_k·R_k → A^(k+1) = R_k·Q_k. Матрица сходится к **квазитреугольной** форме:
- Вещественные λ — на диагонали (1×1-блоки)
- Комплексные пары — в 2×2-блоках на диагонали

**Критерий останова**:
- Столбец m: √(Σ a_im², i>m) < ε → вещественное λ_m = a_mm
- 2×2-блок: |λ^(k) − λ^(k-1)| < ε → комплексная пара (решаем характеристическое уравнение блока)

**Проверка**: tr(A) = Σλ (след матрицы = сумма собственных значений).

**В коде**: матрица 3×3 по варианту + матрица 5×5 с двумя комплексными парами и одним вещественным λ — чтобы продемонстрировать работу с комплексными числами.

## 2.1 — Нелинейное уравнение (одна переменная)

**Уравнение**: f(x) = x³ + x² − x − 0.5 = 0. Ищем положительный корень.

**Локализация**: f(0) = −0.5 < 0, f(1) = 0.5 > 0 → корень на [0, 1].

### Простая итерация

Переписываем f(x) = 0 как x = φ(x), где **φ(x) = x − λ·f(x)**, λ = 0.25.

**Откуда λ = 0.25**: нужно |φ'(x)| < 1. φ'(x) = 1 − λ·f'(x), f'(x) = 3x² + 2x − 1. Так как f''(x) = 6x + 2 > 0, функция f' монотонно возрастает, значит φ' монотонно убывает → **экстремумы φ' только на концах отрезка**. На [0, 1]: f'(0) = −1, f'(1) = 4. Чтобы |1 − 4λ| < 1 и |1 + λ| < 1 не нарушалось, берём 0 < λ < 0.5. Выбираем λ = 0.25.

**q** — коэффициент сжатия, q = max|φ'(x)| на рабочем отрезке. С λ = 0.25 на [0.5, 1]: φ'(0.5) ≈ 0.81, φ'(1) = 0. Значит **q ≈ 0.81**. Он нужен для оценки погрешности: |x* − x_k| ≤ q/(1−q) · |x_k − x_{k-1}|. Чем ближе q к 1, тем медленнее сходимость. В коде используется упрощённый останов |x_{k+1} − x_k| < ε (без множителя q/(1−q) — это более консервативно).

**Сходимость**: линейная, ~20 итераций.

### Метод Ньютона

**Формула**: x_{k+1} = x_k − f(x_k)/f'(x_k). По сути — простая итерация, где λ на каждом шаге автоматически подбирается как 1/f'(x_k).

**Начальное приближение** (формула 2.4): x_0 такое, что f(x_0)·f''(x_0) > 0. f''(x) = 6x + 2 > 0 на [0, 1], f(1) = 0.5 > 0 → x_0 = 1.

**Сходимость**: квадратичная — число верных цифр удваивается на каждой итерации. ~5 итераций.

## 2.2 — Система нелинейных уравнений

**Система** (a = 1):
- f₁ = x₁ − cos(x₂) − 1 = 0
- f₂ = x₂ − lg(x₁ + 1) − 1 = 0

**Локализация** (графическая): рисуем кривые f₁ = 0 и f₂ = 0, их пересечение — начальное приближение ≈ (1.5, 1.3).

### Ньютон для системы

Обобщение: **x^(k+1) = x^(k) − J⁻¹·F**, где J — матрица Якоби (частные производные):

```
J = | 1          sin(x₂)         |
    | -1/((x₁+1)·ln10)    1     |
```

Для 2×2 обратная матрица — по **Крамеру**: dx₁ = det(A₁)/det(J), dx₂ = det(A₂)/det(J). Сходится за ~4 итерации.

### Простая итерация для системы

Выражаем: φ₁ = 1 + cos(x₂), φ₂ = 1 + lg(x₁ + 1). Итерируем: x^(k+1) = φ(x^(k)).

**Условие сходимости**: ||φ'||_∞ < 1. Матрица φ':

```
| 0          -sin(x₂)         |
| 1/((x₁+1)·ln10)   0        |
```

||φ'||_∞ = max(|sin(x₂)|, 1/((x₁+1)·ln10)) ≈ max(0.96, 0.16) = **0.96 < 1** — условие выполнено, но q ≈ 0.96 близко к единице → сходимость медленная, ~150 итераций.

**Оценка погрешности** (формула 2.26): q/(1−q) · max(|Δx₁|, |Δx₂|) < ε.
