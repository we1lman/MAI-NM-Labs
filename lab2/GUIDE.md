# Лаба 2 — Нелинейные уравнения и системы

## Общая идея лабы

В лабе 1 мы решали **линейные** системы (A·x = b). Здесь всё сложнее — уравнения **нелинейные** (есть x^3, cos(x), log(x)). Для них нет формул вроде Гаусса — только итерационные методы: берём начальное приближение и уточняем.

---

## Задача 2.1 — Нелинейное уравнение (`task2_1.py`)

### Постановка (вариант 4)

Уравнение: **f(x) = x^3 + x^2 - x - 0.5 = 0**. Нужно найти положительный корень.

### Шаг 0: Локализация корня

Прежде чем решать — нужно понять, **где примерно** находится корень. Подставляем:
- f(0) = -0.5 < 0
- f(1) = 1 + 1 - 1 - 0.5 = 0.5 > 0

Знак сменился, значит корень на отрезке [0, 1]. Код строит график функции для наглядности.

### Метод 1: Простая итерация (формула 2.7)

**Идея**: переписываем f(x) = 0 в виде x = φ(x), потом итерируем: x_{k+1} = φ(x_k).

В коде: **φ(x) = x - λ·f(x)**, где λ = 0.25. Параметр λ подбирается так, чтобы |φ'(x)| < 1 на отрезке (это **условие сходимости**).

Почему λ = 0.25? Потому что φ'(x) = 1 - λ·f'(x). Производная f'(x) = 3x^2 + 2x - 1. На [0, 1] максимум f' = f'(1) = 4. Чтобы |1 - λ·4| < 1, нужно 0 < λ < 0.5. Берём λ = 0.25 — золотая середина.

**Условие останова**: |x_{k+1} - x_k| < ε.

```
# Строки 59-66 — основной цикл:
x_new = x - lam * f(x)      # φ(x) = x - 0.25·f(x)
diff = abs(x_new - x)       # разница между итерациями
if diff < eps:               # если меньше ε — стоп
    return x_new
```

### Метод 2: Метод Ньютона (формула 2.3)

**Идея**: линеаризуем f(x) в точке x_k (проводим касательную) и находим, где касательная пересекает ось x.

Формула: **x_{k+1} = x_k - f(x_k) / f'(x_k)**

Это как простая итерация, но λ на каждом шаге **подбирается автоматически** как 1/f'(x_k). Поэтому Ньютон сходится **квадратично** (число верных цифр удваивается на каждой итерации), а простая итерация — только линейно.

**Выбор начального приближения** (формула 2.4): x_0 выбираем так, чтобы f(x_0)·f''(x_0) > 0. У нас f''(x) = 6x + 2 > 0 на [0, 1], и f(1) = 0.5 > 0, значит x_0 = 1.

```
# Строки 78-89 — основной цикл:
fx = f(x)                   # значение функции
dfx = df(x)                 # значение производной
x_new = x - fx / dfx        # формула Ньютона
```

### Сравнение методов

| Метод | Порядок сходимости | Итераций |
|-------|-------------------|----------|
| Простая итерация | 1 (линейный) | ~20 |
| Ньютон | 2 (квадратичный) | ~5 |

Ньютон быстрее, но требует вычисления производной. Простая итерация проще, но медленнее.

### Что говорить на защите

- «Локализация: f(0) < 0, f(1) > 0 — корень на [0, 1]»
- «Простая итерация: φ(x) = x - λ·f(x), λ = 0.25 обеспечивает |φ'| < 1»
- «Ньютон: x_{k+1} = x_k - f/f'. Квадратичная сходимость — каждый шаг удваивает число верных цифр»
- «Начальное приближение по Ньютону: f(x_0)·f''(x_0) > 0»
- «Ньютон за 5 итераций, простая итерация за ~20 — видно на графике»

## Задача 2.2 — Система нелинейных уравнений (`task2_2.py`)

### Постановка (вариант 4, a = 1)

Система:
- f_1(x_1, x_2) = x_1 - cos(x_2) - 1 = 0
- f_2(x_1, x_2) = x_2 - lg(x_1 + 1) - 1 = 0

Нужно найти положительное решение (x_1 > 0, x_2 > 0).

### Шаг 0: Локализация (графическая)

Рисуем две кривые:
- Из f_1 = 0: x_1 = 1 + cos(x_2) — «горбик», x_1 от 0 до 2
- Из f_2 = 0: x_2 = 1 + lg(x_1 + 1) — возрастающая кривая

Где они пересекаются — там решение. Из графика: примерно x_1 ≈ 1.5, x_2 ≈ 1.3.

### Метод 1: Метод Ньютона для системы (формулы 2.17-2.20)

Обобщение на систему: вместо деления на f'(x) — умножение на **обратную матрицу Якоби** J^{-1}:

**x^{k+1} = x^k - J^{-1}(x^k) · F(x^k)**

Матрица Якоби — это матрица частных производных:
```
J = | df1/dx1  df1/dx2 |   =   | 1       sin(x2)           |
    | df2/dx1  df2/dx2 |       | -1/((x1+1)·ln10)    1     |
```

Для 2×2-системы обратная матрица считается по **формуле Крамера** (формула 2.20): dx_1 = det(A_1)/det(J), dx_2 = det(A_2)/det(J).

```
# Строки 84-106 — основной цикл:
det_J = j11*j22 - j12*j21          # определитель Якобиана
det_A1 = (-F1)*j22 - j12*(-F2)     # Крамер для dx1
det_A2 = j11*(-F2) - (-F1)*j21     # Крамер для dx2
dx1, dx2 = det_A1/det_J, det_A2/det_J
x1_new, x2_new = x1 + dx1, x2 + dx2
```

**Условие останова** (формула 2.18): max(|dx1|, |dx2|) < ε.

### Метод 2: Простая итерация для системы (формула 2.24)

Из уравнений выражаем x_1 и x_2:
- φ_1(x_1, x_2) = 1 + cos(x_2)
- φ_2(x_1, x_2) = 1 + lg(x_1 + 1)

Итерируем: x_1^{k+1} = φ_1(x_1^k, x_2^k), x_2^{k+1} = φ_2(x_1^k, x_2^k).

**Условие сходимости** (формула 2.25): норма матрицы производных φ' < 1.

Матрица φ':
```
| dφ1/dx1  dφ1/dx2 |   =   | 0          -sin(x2)         |
| dφ2/dx1  dφ2/dx2 |       | 1/((x1+1)·ln10)   0        |
```

||φ'||_∞ = max(|sin(x2)|, 1/((x1+1)·ln10)) ≈ max(0.96, 0.16) = 0.96 < 1. Условие выполнено, но q ≈ 0.96 близко к 1, поэтому сходимость медленная.

**Оценка погрешности** (формула 2.26): q/(1-q) · max(|dx1|, |dx2|) < ε.

```
# Строки 152-154 — основной цикл:
x1_new = 1.0 + math.cos(x2)              # φ_1
x2_new = 1.0 + math.log10(x1 + 1.0)     # φ_2
```

### Что говорить на защите

- «Локализация графическая: рисуем кривые f_1=0 и f_2=0, пересечение — начальное приближение»
- «Ньютон для системы: x^{k+1} = x^k - J^{-1}·F, Якобиан обращаем по Крамеру»
- «Простая итерация: выражаем x_1 = 1+cos(x_2), x_2 = 1+lg(x_1+1)»
- «Условие сходимости итерации: ||φ'|| ≈ 0.96 < 1, но близко к единице — сходимость медленная»
- «Ньютон сходится за ~4 итерации, простая итерация — за ~150»
